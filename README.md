# Lab Sys — University Integrated Laboratory LIMS

Built on Senaite LIMS (open-source)
University: Future University
Location: Indonesia
Floors: 5 (Teaching: 1-3, Research: 4, Office: 5)
Programs: Biomedical, Biotech, Agritech, Food, Pharmacy, Medicine
Status: Planning & Development

## Folders
- backend/ — Python/Django backend + Senaite customizations
- frontend/ — Web interface
- database/ — PostgreSQL schema and seed data
- catalogue/ — Master item catalogue (chemicals, equipment, consumables)
- docs/ — Requirements, architecture, design documents
